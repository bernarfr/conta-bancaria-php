# Sistema Bancário em PHP 8+ (POO Avançada)

Projeto acadêmico demonstrando boas práticas de Programação Orientada a Objetos em PHP.

## Funcionalidades
- Encapsulamento total (atributos e métodos privados)
- Tipagem estrita de propriedades e parâmetros
- Tratamento de exceções em saques
- Separação clara entre lógica de negócio e apresentação
- Formatação monetária brasileira

## Como executar
```bash
php banco.php
